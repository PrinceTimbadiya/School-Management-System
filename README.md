# 🎓 School Management System (C# WinForms)

## 👨‍💻 About Project

This is a complete **School Management System desktop application** built using **C# (.NET WinForms)** and **SQL Server (LocalDB)**.

It helps manage students, staff, fees, attendance, and academic data efficiently.

---

## 🚀 Features

* 🔐 Login System
* 👨‍🎓 Student Management
* 👨‍🏫 Staff Management
* 🏫 Department & Class Management
* 📚 Subject & Subject Allotment
* 💰 Fees Management
* 📊 Reports (Crystal Reports)
* 🔍 Search System

---

## 🛠 Tech Stack

* C# (.NET WinForms)
* SQL Server (LocalDB)
* Visual Studio 2012
* Crystal Reports

---

## ⚙️ How to Run

1. Open project in Visual Studio
2. Attach `school.mdf` if needed
3. Press **F5** to run
